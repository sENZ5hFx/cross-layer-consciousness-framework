# Security Policy

## Owner

Haley Ann Bird — heyhaleybird@gmail.com

## Supported Scope

This repository contains theoretical framework code, prototype implementations, benchmarks,
and academic preprint content. Security-sensitive surfaces include:

- Any executable prototype code in `prototype/`
- Any data pipelines or scripts in `scripts/`
- Any external API integrations or credential references

## Reporting a Vulnerability

**Do not** open a public GitHub issue for security vulnerabilities.

Email: **heyhaleybird@gmail.com**
Subject: `[SECURITY] cross-layer-consciousness-framework — <description>`

Include:
- Vulnerability type and affected component
- File paths and branch/commit where issue exists
- Steps to reproduce
- Potential impact assessment
- Suggested remediation (optional)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 5 days
- **Resolution timeline:** Provided after assessment
- **Public disclosure:** After fix + 30 days

## Safe Harbor

Haley Ann Bird will not pursue legal action against researchers who responsibly disclose
vulnerabilities without exploiting or publicly disclosing them before remediation.
